Browser Usage Statistics

Test 1: Setting X-Frame-Options to DENY or SAMEORIGIN will make this test pass because less than 0.2% of the market use browsers that do not support these options. Using ALLOW-FROM causes this test to fail, because ALLOW-FROM is not supported in Chrome, which makes up about 70% of the browser market share.

Warning for Content-Security-Policy users: more browsers support X-Frame-Options. Verify that the Content-Security-Policy Support covers your user base.
